# Notes Journal

Приложение за бележки с възможност за прикачване на снимки.

## Идея на проекта

Notes Journal е приложение за водене на бележки. Потребителите могат да създават, редактират и изтриват бележки, както и да прикачват снимки към тях чрез камерата на устройството.

## Как работи

**Основни функции:**

- Създаване на бележка с title и content
- Редактиране на съществуващи бележки
- Изтриване на бележки (дълго натискане)
- Прикачване на снимка от камерата
- Преглед на снимка на цял екран
- Данните се запазват локално в Room база данни

**Съхранение на данни:**

Всяка бележка съдържа:
- ID (автоматично генерирано)
- Заглавие
- Съдържание
- Дата на създаване/редакция
- Път към снимка (опционално)

## Архитектура

Приложението използва **MVVM** архитектура с Repository слой:

- **UI слой** - Fragments и Adapters
- **ViewModel слой** - NoteViewModel (държи състоянието на UI)
- **Repository слой** - NoteRepository (абстракция над данните)
- **Data слой** - Room Database, DAO, Entity

**Използвани библиотеки:**
- Room - локална база данни
- Navigation Component - навигация между екрани
- ViewModel & Flow - управление на състоянието
- Coroutines - асинхронни операции
- Material Design 3 - UI компоненти

## Потребителски сценарии

**Създаване на бележка:**
1. Натисни бутона (+)
2. Въведи заглавие и съдържание
3. (Опционално) Натисни "Take Photo" за снимка
4. Натисни "Save"

**Редактиране на бележка:**
1. Натисни върху бележка от списъка
2. Промени данните
3. Натисни "Save"

**Изтриване на бележка:**
1. Задръж дълго върху бележка
2. Потвърди изтриването

**Преглед на снимка:**
1. Отвори бележка със снимка
2. Натисни върху снимката за цял екран
3. Натисни отново за затваряне

## Инсталация

**Изисквания:**
- Android Studio
- Android SDK 35
- JDK 11+

**Стъпки:**
1. Клонирай репозиторито
2. Отвори в Android Studio
3. Изчакай Gradle sync
4. Build → Make Project
5. Run → Run 'app'

**APK инсталация:**
1. Свали APK от `/apk` папката
2. Инсталирай на устройство
3. Стартирай приложението
